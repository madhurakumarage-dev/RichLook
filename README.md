# Rich Look - Fashion E-Commerce App

A modern, production-ready fashion e-commerce mobile application built with Flutter.

## Overview

**Rich Look** provides a premium shopping experience with a clean, dynamic, and responsive UI. The app follows a clean architecture separating screens, models, and providers, and utilizes the `Provider` package for state management.

## Features

*   **Beautiful UI/UX:** Built with a custom design system, including tailored typography (Google Fonts) and a cohesive color palette.
*   **Product Discovery:** Browse products with category filtering and a responsive grid layout.
*   **Product Details:** View detailed product information, select sizes/colors, and add items to the cart.
*   **Cart Management:** Reactive cart state updates with seamless checkout flow.
*   **User Profile:** Manage user details and preferences in a clean profile interface.
*   **Smooth Animations:** Engaging micro-animations (like cart badges) for a polished feel.

## Tech Stack

*   **Framework:** Flutter
*   **State Management:** `provider`
*   **Fonts:** `google_fonts`

## Getting Started

To run this project locally:

1.  **Ensure Flutter is installed:** Follow the instructions on the [official Flutter website](https://docs.flutter.dev/get-started/install).
2.  **Clone the repository.**
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
